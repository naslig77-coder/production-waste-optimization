# Definición del Problema

## Contexto

En los procesos productivos industriales es común que exista pérdida de materia prima durante la operación.  
Este desperdicio impacta directamente:

- Costos de producción
- Eficiencia operativa
- Rentabilidad del proceso

Sin una medición estructurada, estas pérdidas suelen pasar desapercibidas o no ser gestionadas correctamente.

---

## Problema

Actualmente, la información relacionada con:

- Desperdicios generados
- Peso estándar de producción
- Máquinas involucradas
- Departamentos

se encuentra distribuida en múltiples fuentes de datos sin un proceso claro de integración y análisis.

Esto impide:

- Identificar qué máquinas generan mayor pérdida
- Detectar instalaciones con baja eficiencia
- Medir el impacto real del desperdicio sobre la producción

---

## Objetivo del Proyecto

Diseñar un proceso ETL que permita:

1. Integrar datos de producción y desperdicio
2. Estandarizar la información
3. Generar datasets limpios para análisis

Con el fin de:

- Medir el porcentaje de desperdicio
- Identificar fuentes principales de pérdida
- Facilitar la toma de decisiones operativas

---

## Preguntas de Negocio a Resolver

Este proyecto busca responder:

- ¿Qué máquinas generan mayor desperdicio?
- ¿Qué departamentos presentan el mayor desperdicio?
- ¿Cuál es el impacto del desperdicio sobre la cantidad de kilos entregada?
- ¿Existen patrones que permitan optimizar el proceso?

---

## Valor Esperado

El análisis permitirá:

- Reducir pérdidas productivas
- Identificar los procesos con mayor desviación sobre el estándar
- Optimizar uso de materia prima
- Apoyar decisiones basadas en datos
